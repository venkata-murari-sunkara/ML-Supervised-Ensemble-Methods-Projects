**ML Project 1 - Predicting Hotel Booking Cancellations (InnHotels Group)**
**Project Overview**
This project analyzes **hotel booking behavior** and **builds predictive models** to determine whether a booking will be **canceled or not canceled**. Using a **dataset of 36,275 bookings**, the project explores customer demographics, booking patterns, pricing behavior and special requests to uncover the key drivers of cancellations.
The final output includes EDA, feature engineering, model comparison, and actionable business recommendations for InnHotels Group.


**ML Project 2 - Visa Approval Prediction (EasyVisa)
Project Overview**
This project builds **machine learning models** to predict whether a **visa application will be Certified or Denied**. Using a **dataset of 25,480 applications**, the project explores applicant demographics, employer characteristics, wages, and job requirements to uncover patterns influencing visa outcomes.
The project includes **EDA, preprocessing, oversampling/undersampling, multiple ML models, and hyperparameter tuning**.
